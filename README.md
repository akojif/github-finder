# Github Finder

A better look at your GitHub Profile! With charts!

![demo](https://raw.githubusercontent.com/akojif/githubfinder/master/static/og.png)

Built with:

- [Next.js](https://nextjs.org/)
- [GitHub Polyglot](https://github.com/IonicaBizau/node-gh-polyglot)
- [Chart.js](https://www.chartjs.org/)
- [React Flip Move](https://github.com/joshwcomeau/react-flip-move)
- [Styled Components](https://www.styled-components.com/)

Inspired by [Brittany Chiang](https//:github.com/bchiang7)

## Getting Started

1. Install dependencies

   ```bash
   yarn
   ```

2. Fire up the server and watch files

   ```bash
   npm run dev
   ```

## Deployment

1. Install the Now CLI

   ```bash
   npm install -g now
   ```

2. Deploy!

   ```bash
   now
   ```
